Vault RMS Injury Tracker

A web application to help firefighters expedite the injury form filling process by automating applicable portions of the process. A user will log in and will be able to start filling out pre-populated applications using data from a master bank. The final product is a completed form in the required PDF format.

  • MongoDB
  • ExpressJS
  • AngularJS
  • NodeJs
  • PassportJS
  • jsPDF

Express ER

A tablet application that expedites the checkin process within an emergency room by supplanting a receptionist. A user can use the tablet application to fill out personal information and symptoms for the hospital visit. The application uses a priority algorithm to alert the staff which patients need to be seen first

  • SQL
  • ExpressJS
  • AngularJS
  • NodeJs
  • Twilio
